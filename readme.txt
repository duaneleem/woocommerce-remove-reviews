Disable the reviews tab across WooCommerce.

== Description ==

Looking to disable the review tab from your WooCommerce product?  You can activate and deactivate that feature with this plugin.

A few notes about this plugin:

* No admin menu. Just do the next steps to enable/disable WooCommerce reviews.
* To disable the reviews tab, enable this plugin.
* To enable the reviews tab, simply disable this plugin.
